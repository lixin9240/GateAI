### EdgeToken HMAC 认证

| 变更 | 内容 |
|------|------|
| 新增 | `edge_nodes` 表新增 `api_secret` 字段，新建节点时自动生成 |
| 重写 | `VerifyEdgeToken` 中间件 — HMAC-SHA256 签名校验 + 时间戳防重放 + Nonce 防重放 |
| 认证格式 | `Authorization: EdgeToken <edge_id>.<timestamp>.<nonce>.<signature>` |
| 新增路由组 | `/v1/edge/*` 独立于 `auth:api`，不再要求用户登录 |

**分层**：Service → `App\Services\Wjc\GateInterlockService`，Controller → `App\Http\Controllers\Api\Wjc\GateInterlockController`，Request → `App\Http\Requests\Wjc\GateInterlockRequest`。

**签名算法**：
```
payload = edge_id + "\n" + timestamp + "\n" + nonce + "\n" + METHOD + "\n" + path + "\n" + body
signature = HMAC-SHA256(payload, api_secret)
```
校验规则：时间戳偏差 > 300s 拒绝，Nonce 重复使用拒绝（缓存 1h），签名不匹配拒绝。
