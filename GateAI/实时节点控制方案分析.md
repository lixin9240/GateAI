## 三、水力学精确推导

### 3.1 符号定义

```
Z(t)       : 时刻 t 的坝前水位 (m)
Q_in(t)    : 时刻 t 的入库流量 (m³/s)
Q_out(t)   : 时刻 t 的出库流量 (m³/s)
Q_eco      : 生态基流 (m³/s)，必须保证的最小下泄
A(Z)       : 水位 Z 对应的库区水面面积 (m²)
gᵢ(t)      : 时刻 t 第 i 号闸门的开度 (0~1)
Q_max,ᵢ    : 第 i 号闸门的最大泄流能力 (m³/s)
Δt         : 时间步长 (s)，控制台的采样间隔
S_down(t)  : 下游水位 (m)
R(t)       : 时刻 t 的降雨量 (mm/h)
```

### 3.2 出库流量公式（闸门水力特性）

```
Q_out(t) = Q_eco + Σᵢ [ gᵢ(t) × Q_max,ᵢ × √(Z(t) - S_down(t)) / √H_ref ]

其中:
  H_ref = Z_normal - S_down_normal (参考水头)
  Q_max,ᵢ: 闸门1(溢洪道)=300, 闸门2(泄洪洞)=200, 闸门3(发电引水)=250
```

### 3.3 水量平衡微分方程

```
dV(t)/dt = Q_in(t) - Q_out(t)
A(Z) = A_dead + (A_normal - A_dead) × (Z - Z_dead) / (Z_normal - Z_dead)
dZ/dt = [ Q_in(t) - Q_out(t) ] / A(Z(t))

离散化:
Z(t+Δt) = Z(t) + [ Q_in(t) - Q_out(t) ] × Δt / A(Z(t))
```

### 3.4 多节点级联传播方程

```
Q_in(n, t) = Q_out(m, t - τ_m→n) + Q_lateral(n, t)

τ_m→n = d_m→n / v_flow (水流传播延迟)
```

| 上游 | 下游 | 距离 | 流速 | 延迟 τ |
|------|------|------|------|:--:|
| 三峡 | 溪洛渡 | 156 km | 2.5 m/s | ~17.3 h |
| 溪洛渡 | 向家坝 | 184 km | 3.0 m/s | ~17.0 h |
| 向家坝 | 示范 | 140 km | 2.0 m/s | ~19.4 h |

### 3.5 降雨-径流关系

```
ΔQ_in(t) = R(t) × C_runoff × A_catchment / 3600

其中: C_runoff=0.55 (三峡库区), A_catchment=100km²
```

### 3.6 下游水位-出库流量关系

```
S_down(t) = S_down,base + α × (Q_out(t) - Q_out,base)^(2/3)

α = n / (B × √J)  (n=0.030, B=800m, J=0.0005)
```

```php
<?php
namespace App\Services\GYZ;
use App\Models\Reservoir;
use App\Models\EdgeNode;

/**
 * 水力学计算引擎
 * 公式 1: outflow()         闸门出库流量,对应第三章 3.2 式
 * 公式 2: surfaceArea()     水位-面积梯形插值,对应第三章 3.3 式
 * 公式 3: waterBalance()    水量平衡微分,对应第三章 3.3 式
 * 公式 4: rainfallToInflow()降雨→入库转换,对应第三章 3.5 式
 * 公式 5: downstreamLevel() 下游水位-流量关系,对应第三章 3.6 式
 * 综合:   predict()         串联 ①~⑥ 输出 6h 预测序列
 */
class HydraulicCalculator
{
    private const GATE_MAX_DISCHARGE = [300.0, 200.0, 250.0];
    private const ECO_FLOW_MIN        = 20.0;
    private const RUNOFF_COEFFICIENT  = 0.55;
    private const CATCHMENT_AREA_KM2  = 100.0;
    private const RIVER_WIDTH_M       = 800.0;
    private const RIVER_SLOPE         = 0.0005;
    private const MANNING_N           = 0.030;
    private array $reservoirCache = [];

    /**
     * 公式 1: 闸门出库流量 (第三章 3.2)
     * Q_out = Q_eco + Σ gᵢ × Q_max,ᵢ × √(Z-S_down) / √H_ref
     */
    public function outflow(array $gateOpenings, float $upstreamLevel, float $downstreamLevel, float $referenceHead): float
    {
        $head = max(0.1, $upstreamLevel - $downstreamLevel);
        $ref  = max(0.1, $referenceHead);
        $sqrtRatio = sqrt($head) / sqrt($ref);
        $discharge = self::ECO_FLOW_MIN;
        foreach ($gateOpenings as $i => $g) $discharge += $g * self::GATE_MAX_DISCHARGE[$i] * $sqrtRatio;
        return round($discharge, 2);
    }

    /**
     * 公式 2: 水位-面积梯形插值 (第三章 3.3)
     * A(Z) = A_dead + (A_normal - A_dead) × (Z - Z_dead) / (Z_normal - Z_dead)
     */
    public function surfaceArea(int $reservoirId, float $level): float
    {
        $r = $this->getReservoir($reservoirId);
        if (! $r) return 15_000_000;
        $Z_dead   = (float) $r['dead_water_level'];
        $Z_normal = (float) $r['normal_water_level'];
        $avgDepth = max(1.0, $Z_normal - $Z_dead);
        $A_normal = (float) $r['total_capacity'] * 10000.0 / $avgDepth;
        $A_dead   = $A_normal * 0.30;
        $ratio    = max(0.0, min(1.0, ($level - $Z_dead) / $avgDepth));
        return $A_dead + ($A_normal - $A_dead) * $ratio;
    }

    /**
     * 公式 3: 水量平衡 → 水位变化 (第三章 3.3)
     * dZ = (Q_in - Q_out) × Δt / A(Z)
     */
    public function waterBalance(float $inflow, float $outflow, float $area, int $dt = 3600): float
    {
        return ($inflow - $outflow) * $dt / max(1.0, $area);
    }

    /**
     * 公式 4: 降雨→入库流量 (第三章 3.5)
     * ΔQ_in = R × C_runoff × A_catchment / 3600
     */
    public function rainfallToInflow(float $rainfall): float
    {
        return $rainfall > 0 ? $rainfall * self::RUNOFF_COEFFICIENT * self::CATCHMENT_AREA_KM2 / 3600.0 : 0.0;
    }

    /**
     * 公式 5: 下游水位-流量关系 (第三章 3.6)
     * S_down = S₀ + α × (Q - Q₀)^(2/3),  α = n/(B√J)
     */
    public function downstreamLevel(float $outflow, float $baseLevel, float $baseFlow): float
    {
        $alpha = self::MANNING_N / (self::RIVER_WIDTH_M * sqrt(self::RIVER_SLOPE));
        $delta = $outflow - $baseFlow;
        if (abs($delta) < 1.0) return $baseLevel;
        $sign = $delta > 0 ? 1.0 : -1.0;
        return round($baseLevel + $sign * $alpha * pow(abs($delta), 2.0 / 3.0), 3);
    }

    /**
     * 综合预测: 串联公式 ①→⑥，输出 6h 预测序列
     *
     * FOR h=1 TO 6:
     *   ① 降雨修正 Q_in
     *   ② 计算当前水面面积 A(Z)
     *   ③ 计算当前出库流量 Q_out
     *   ④ 水量平衡 → 水位变化 dZ
     *   ⑤ 下游水位回算
     *   ⑥ Z←Z+dZ，记录
     */
    public function predict(int $rid, array $gates, float $Z, float $S, float $Qin, float $rain, float $Href, int $hours=6): array
    {
        $baseQ  = $Qin;
        $dt     = 3600;
        $levels = []; $inflows = [];
        for ($h=1; $h<=$hours; $h++) {
            $Qin += $this->rainfallToInflow($rain);
            $A    = $this->surfaceArea($rid, $Z);
            $Qout = $this->outflow($gates, $Z, $S, $Href);
            $dZ   = $this->waterBalance($Qin, $Qout, $A, $dt);
            $S    = $this->downstreamLevel($Qout, $S, $baseQ);
            $Z   += $dZ;
            $levels[]  = round($Z, 2);
            $inflows[] = round($Qin, 1);
            if ($rain <= 0) $Qin *= 0.97;
        }
        $danger = $this->getReservoir($rid)['flood_limit_level'] ?? 190;
        $peak   = max($levels);
        $safety = $peak > $danger ? 'danger' : ($peak > $danger - 3 ? 'warning' : 'safe');
        return ['predicted_levels'=>$levels,'predicted_inflows'=>$inflows,'predicted_peak_level'=>$peak,'safety_flag'=>$safety,'risk_probability'=>$peak>$danger?min(1,($peak-$danger)/10):0.05,'physics_passed'=>$peak<$danger+5,'decision_mode'=>$safety==='danger'?'L1_MANUAL':'L3_AUTO','confidence'=>0.65];
    }

    /**
     * 下游传播: ΔZ_down ≈ ΔQ_up × 3600 / A_down, 延迟 τ = d/v (第三章 3.4)
     */
    public function downstreamImpact(int $fromNodeId, array $newG, array $oldG, float $Z, float $S, float $Href): array
    {
        $topo = [1=>['n'=>[4,5],'h'=>17.3,'km'=>156,'v'=>2.5],2=>['n'=>[4,5],'h'=>17.3,'km'=>156,'v'=>2.5],3=>['n'=>[4,5],'h'=>17.3,'km'=>156,'v'=>2.5],4=>['n'=>[6],'h'=>17.0,'km'=>184,'v'=>3.0],5=>['n'=>[6],'h'=>17.0,'km'=>184,'v'=>3.0],6=>['n'=>[7,8],'h'=>19.4,'km'=>140,'v'=>2.0]];
        $t = $topo[$fromNodeId]??null; if(!$t)return['has_downstream'=>false];
        $oldQ = $this->outflow($oldG,$Z,$S,$Href); $newQ = $this->outflow($newG,$Z,$S,$Href); $dQ = $newQ-$oldQ;
        $details = []; foreach($t['n'] as $did){$dn=EdgeNode::find($did); $dr=$dn?\App\Models\Reservoir::find($dn->reservoir_id):null; $dA=$dr?(float)($dr->total_capacity*10000/max(1,$dr->normal_water_level-$dr->dead_water_level)):10_000_000; $details[]=['node_id'=>$did,'name'=>$dn?->name??'?','est_level_change_m'=>round($dQ*3600/max(1,$dA),3)];}
        return['has_downstream'=>true,'affected_nodes'=>$t['n'],'delay_hours'=>$t['h'],'distance_km'=>$t['km'],'flow_velocity_ms'=>$t['v'],'delta_Q_m3s'=>round($dQ,1),'message'=>"约{$t['h']}h后下游可观测(ΔQ={$dQ}m³/s)",'details'=>$details];
    }

    private function getReservoir(int $id): ?array {
        if(!isset($this->reservoirCache[$id])){$r=Reservoir::select('id','dead_water_level','normal_water_level','flood_limit_level','total_capacity')->find($id);$this->reservoirCache[$id]=$r?$r->toArray():null;}
        return $this->reservoirCache[$id];
    }
}
```

```php
<?php
namespace App\Services\GYZ;

class GateInterlockPreviewService
{
    public function check(array $openings, array $prevOpenings = null): array
    {
        [$g1, $g2, $g3] = $openings;
        $prev = $prevOpenings ?? $openings;
        $rejected = []; $warnings = [];

        if ($g1 > 80 && $g3 > 50) {
            $rejected[] = ['rule'=>'spillway_intake_mutex','msg'=>'溢洪道>80%时发电闸<=50%'];
        }
        if ($prev) {
            $inc = 0;
            for ($i=0;$i<3;$i++) if ($openings[$i]-$prev[$i]>30) $inc++;
            if ($inc>=2) $rejected[] = ['rule'=>'downstream_impact','msg'=>'禁止两个闸门同时大幅增开(>30%)'];
        }
        if (abs($g1-$g2)>40) $warnings[] = ['rule'=>'symmetry','msg'=>'两侧开度差>40%，建议对齐'];
        if ($g1+$g2+$g3<5) $rejected[] = ['rule'=>'min_discharge','msg'=>'总开度不能低于5%'];

        return ['passed'=>empty($rejected),'rejected'=>$rejected,'warnings'=>$warnings];
    }
}
```

```php
                '互锁拒绝：' . $interlock['rejected'][0]['msg'], ['interlock'=>$interlock]);
        }

        // 3. AI 推理
        $aiResult = HydropowerService::infer([
            'upstream_level'=>$v['upstream_level'],'downstream_level'=>$v['downstream_level'],
            'inflow'=>$v['inflow'],'rainfall'=>$v['rainfall']??0,
            'gate1_opening'=>$gateNew[0],'gate2_opening'=>$gateNew[1],'gate3_opening'=>$gateNew[2],
        ]);

        $fallback = false;
        if (! $aiResult) {
            $aiResult = $this->physicsEstimate($v, $g1, $g2, $g3, $node);
            $fallback = true;
        }

        // 4. 下游传播
        $downstream = $this->estimateDownstream($node, $gateNew, $gateOld, $v);

        // 5. 安全判定
        $safety = $this->assessSafety($aiResult, $interlock);

        return Result::success($fallback?'预览完成（降级估算）':'预览完成', [
            'fallback'=>$fallback,'interlock'=>$interlock,'ramp_plan'=>$ramp,
            'can_execute'=>$safety['can_execute'],'safety_level'=>$safety['level'],'safety_msg'=>$safety['msg'],
            'impact'=>[
                'gate_openings'=>$gatePct,'current_openings'=>$gateOldPct,
                'predicted_levels'=>$aiResult['predicted_levels']??[],
                'predicted_inflows'=>$aiResult['predicted_inflows']??[],
                'predicted_peak'=>$aiResult['predicted_peak_level']??null,
                'safety_flag'=>$aiResult['safety_flag']??'unknown',
                'risk_probability'=>$aiResult['risk_probability']??null,
                'physics_passed'=>$aiResult['physics_passed']??false,
            ],
            'downstream_impact'=>$downstream,
        ]);
    }

    /**
     * 确认执行
     * POST /api/v1/monitor/node-control/execute
     */
    public function execute(Request $request): JsonResponse
    {
        $v = $request->validate([
            'edge_node_id'   =>'required|integer|exists:edge_nodes,id',
            'gate_openings'  =>'required|array|size:3',
            'gate_openings.*'=>'numeric|min:0|max:100',
            'version'        =>'required|integer',
            'reason'         =>'nullable|string|max:255',
            'force'          =>'nullable|boolean',
        ]);

        $node = EdgeNode::findOrFail($v['edge_node_id']);
        $this->checkVersion($node, (int)$v['version']);
        $newG = array_map('floatval', $v['gate_openings']);
        $oldG = [(float)$node->gate1_opening,(float)$node->gate2_opening,(float)$node->gate3_opening];

        if (! $request->boolean('force')) {
            $safety = HydropowerService::infer([
                'upstream_level'=>180,'downstream_level'=>118,'inflow'=>350,
                'gate1_opening'=>$newG[0]/100,'gate2_opening'=>$newG[1]/100,'gate3_opening'=>$newG[2]/100,
            ]);
            if (($safety['safety_flag']??'safe')==='danger') {
                return Result::error(ResponseCode::STATUS_CANNOT_OPERATE,
                    'danger级别，需传force:true确认执行');
            }
        }

        $node->update([
            'gate1_opening'=>$newG[0],'gate2_opening'=>$newG[1],'gate3_opening'=>$newG[2],
            'control_mode'=>'L1_MANUAL','version'=>$node->version+1,
        ]);

        $this->logAudit($node->id,'execute',$oldG,$newG,null);

        try {
            broadcast(new \App\Events\CommandIssued((string)$node->id,[
                'gate_openings'=>$newG,'reason'=>$v['reason']??'操作员手动调闸',
            ]));
        } catch (\Exception $e) {}

        return Result::success("已下发至 {$node->name}",['version'=>$node->version]);
    }

    /**
     * 全站场景切换
     * POST /api/v1/monitor/node-control/scenario
     */
    public function scenario(Request $request): JsonResponse
    {
        $v = $request->validate([
            'reservoir_id'=>'required|integer|exists:reservoirs,id',
            'scenario'=>'required|string|in:flood_defense,drought_conserve,restore_auto',
        ]);

        if ($v['scenario']==='restore_auto') {
            EdgeNode::where('reservoir_id',$v['reservoir_id'])->update(['control_mode'=>'L3_AUTO']);
            return Result::success('已恢复全自动模式');
        }

        $openings = $v['scenario']==='flood_defense' ? [100,100,100] : [5,5,5];
        $nodes = EdgeNode::where('reservoir_id',$v['reservoir_id'])->get();

        foreach ($nodes as $node) {
            $old = [(float)($node->getOriginal('gate1_opening')??0),(float)($node->getOriginal('gate2_opening')??0),(float)($node->getOriginal('gate3_opening')??0)];
            $node->update(['gate1_opening'=>$openings[0],'gate2_opening'=>$openings[1],'gate3_opening'=>$openings[2],'control_mode'=>'L1_MANUAL','version'=>$node->version+1]);
            $this->logAudit($node->id,'scenario',$old,$openings,null);
            try { broadcast(new \App\Events\CommandIssued((string)$node->id,['gate_openings'=>$openings,'reason'=>$v['scenario']==='flood_defense'?'全站防洪':'全站保供'])); } catch (\Exception $e) {}
        }

        return Result::success("已下发至 {$nodes->count()} 个节点");
    }

    // ═══════════════════════════════════════
    private function checkVersion($node,int $ver):void{if((int)$node->version!==$ver)throw new BusinessException('已被他人修改，请刷新',ResponseCode::DATA_VERSION_CONFLICT);}
    private function checkRampRate($old,$new):array{$d=max(abs($new[0]-$old[0]),abs($new[1]-$old[1]),abs($new[2]-$old[2]));$s=(int)ceil($d/self::MAX_GATE_CHANGE_PER_HOUR_PCT);return['requires_ramping'=>$s>1,'steps'=>$s,'hours'=>$s,'per_step_pct'=>round($d/max($s,1),1)];}

    /**
     * 降级估算：调用 HydraulicCalculator 纯物理公式 (第三章 3.2~3.6)
     * 仅在 HydropowerService::infer() 返回 null 时使用
     */
    private function physicsEstimate($v, $g1, $g2, $g3, $node): array
    {
        $rid  = (int) $node->reservoir_id;
        $Z    = (float) $v['upstream_level'];
        $S    = (float) $v['downstream_level'];
        $Qin  = (float) $v['inflow'];
        $rain = (float) ($v['rainfall'] ?? 0);
        $res  = \App\Models\Reservoir::find($rid);
        $Href = $res ? (float) $res->normal_water_level - 118.0 : 57.0;

        return $this->hydraulic->predict(
            $rid,
            [$g1 / 100, $g2 / 100, $g3 / 100],
            $Z, $S, $Qin, $rain, $Href
        );
    }

    /**
     * 下游传播预估：调用 HydraulicCalculator::downstreamImpact()
     */
    private function estimateDownstream($node, $gateNew, $gateOld, $v): array
    {
        $res  = \App\Models\Reservoir::find($node->reservoir_id);
        $Href = $res ? (float) $res->normal_water_level - 118.0 : 57.0;
        $Z    = (float) ($v['upstream_level'] ?? 180);
        $S    = (float) ($v['downstream_level'] ?? 118);

        return $this->hydraulic->downstreamImpact(
            (int) $node->id, $gateNew, $gateOld, $Z, $S, $Href
        );
    }

    private function assessSafety($ai,$interlock):array{
        $f=$ai['safety_flag']??'danger';$p=$ai['physics_passed']??false;$r=$ai['risk_probability']??1.0;$w=!empty($interlock['warnings']);
        if($f==='danger'||!$p||$r>0.8)return['can_execute'=>false,'level'=>'red','msg'=>'安全风险过高，拒绝执行'];
        if($f==='warning'||$r>0.5)return['can_execute'=>true,'level'=>'orange','msg'=>'风险偏高（'.round($r*100).'%），谨慎操作'];
        if($w)return['can_execute'=>true,'level'=>'yellow','msg'=>'互锁警告，确认后执行'];
        return['can_execute'=>true,'level'=>'green','msg'=>'安全，可执行'];
    }

    private function logAudit($nid,$act,$bf,$af,$im):void{NodeControlLog::create(['edge_node_id'=>$nid,'operator_id'=>auth('api')->id(),'action_type'=>$act,'gate1_before'=>$bf[0],'gate1_after'=>$af[0],'gate2_before'=>$bf[1],'gate2_after'=>$af[1],'gate3_before'=>$bf[2],'gate3_after'=>$af[2],'impact_result'=>$im?json_encode($im):null,'control_mode_before'=>'L3_AUTO','control_mode_after'=>'L1_MANUAL','operated_at'=>now()]);}
}
```
